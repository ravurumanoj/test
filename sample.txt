custom-ghep-lang-and-VS Code Extension Setup Guide

A step-by-step guide to create the custom-ghep-lang-api' VS Code extension from scratch. This extension starts a local Express.js HTTP server (port 3000) inside VS Code that proxies requests to Github Copilot's Language Model API.
